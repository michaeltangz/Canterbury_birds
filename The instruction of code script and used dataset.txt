2. Join the grid data with the filtered bird dataset and clip the grid data to cantebury region
join_data.ipynb ==> cbirds_grid.gpkg
**[used dataset]**
  |_ statsnz-new-zealand-grid-1-kilometre-2023-SHP.zip
  |_ statsnz-regional-council-2023-clipped-generalised-SHP.zip
  |_ grid_1km_canterbury.shp
  |_ cbirds_filtered.csv

3. Extract the landsurface day temperature from the stalelite data, clip it to cantebury region and calucate the average temperature;
average_temp_new.ipynb ===> cantburry_average_temp_1.tif
**[used dataset]**
  |_ cantbury_region.shp
